# 🚚 Logistická platforma (Bakalářská práce)

Webová platforma pro vyhledávání a zadávání přepravních zakázek.
Tento projekt je vyvíjen jako praktická část bakalářské práce.

## 🛠️ Technologie

- **Frontend:** React, Vite, Tailwind CSS, Leaflet (Mapy)
- **Backend:** Node.js, Express
- **Databáze:** PostgreSQL + PostGIS (prostorová data)
- **Infrastruktura:** Docker & Docker Compose

## 🚀 Jak spustit projekt lokálně

1. Ujistěte se, že máte nainstalovaný a spuštěný **Docker**.
2. V kořenové složce projektu spusťte příkaz:
   ```bash
   docker compose up --build
   ```
